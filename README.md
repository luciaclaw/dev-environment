# dev-environment

Local development environment for the Lucia privacy-preserving AI agent platform.

## Features

- One-command Docker Compose setup
- Mock CVM for local agent development
- Skill sandbox emulator
- Test inference endpoint
- Mock attestation service

## Getting Started

```bash
docker compose up
```

## About Lucia

Lucia is a privacy-preserving AI agent platform that delivers autonomous agent capabilities with hardware-enforced privacy via Trusted Execution Environments (TEE). Learn more at [luciaclaw.com](https://luciaclaw.com).
